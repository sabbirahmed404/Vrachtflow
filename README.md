# Vrachtflow

## Description

Vrachtflow is a web application that allows users to manage their tasks and projects.

## Tech Stack

- Next.js
- Tailwind CSS
- TypeScript
- Prisma
- PostgreSQL
- AuthJS
- React Query
- React Hook Form
- React Router
- React Icons
- React Select
- React Dropzone
- React Date Picker
- React Toastify
- React Hook Form

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```
